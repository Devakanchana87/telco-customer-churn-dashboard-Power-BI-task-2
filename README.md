# Customer Retention & Churn Analysis Dashboard — Power BI
 
## Overview
Customer retention and churn analysis dashboard built using Power BI, covering churn drivers, customer segments, and retention insights for a telecom company.
 
## Tools Used
- Power BI Desktop
- Power Query
## Telco Customer Churn Dataset
- 7,043 customers | 21 features
- Target variable: Churn (Yes/No)
- Source: Telco Customer Churn Dataset (Kaggle)
## Key Insights
1. Contract type is the strongest churn driver — month-to-month customers churn at ~42%, vs. ~3% for two-year contracts
2. Fiber optic customers churn the most despite higher charges, suggesting service or pricing dissatisfaction
3. Electronic check users have the highest churn (~45%) and lowest retention — automatic payments correlate with loyalty
4. The first 12 months are the riskiest period for customer loss
5. Senior citizens on month-to-month plans churn at ~55% — the highest-risk segment overall
6. Overall retention rate is 73.46%
## Recommendations
1. Incentivize longer-term contracts with discounts for switching from month-to-month
2. Investigate Fiber Optic service quality and reliability issues
3. Promote automatic payment methods (bank transfer/credit card) over electronic check
4. Launch onboarding and loyalty programs targeting first-year customers
5. Offer free trials of OnlineSecurity/TechSupport add-ons to increase stickiness
